swottle
===============
Sea Web Bottle

setup
------------------

1.  install Sinatra and dependencies (see below)
2.  copy **config.rb-dist** to **config.rb** and edit it
3.  launch application: 
        ruby swottle.rb
4.  go to http://localhost:4567
