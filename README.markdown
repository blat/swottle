Sea Web Bottle
===============

This tools was based on an original idea of [taziden](http://www.flexiden.org).

Dependances
------------------
* sinatra
    * rack
* haml
* dm-core
    * addressable
    * data_objects
    * extlib
* rake
* rspec
* cucumber
    * builder
    * term-ansicolor
    * diff-lcs
    * polyglot
* hoe   
    * rubyforge
* launchy
* dm-validations
* dm-timestamps
* dm-aggregates
* do_sqlite3

Setup
------------------

1.  Install ruby and all dependencies (see above)

2.  Copy `config.rb-dist` to `config.rb` and edit it

3.  Launch application: 

        ruby swottle.rb

4.  Go to [http://localhost:4567](http://localhost:4567)
